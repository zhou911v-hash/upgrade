# 智能楼宇照明监控系统用户操作手册

## 1. 系统启动
1. 打开生成后的 WPF 桌面程序。
2. 在登录页输入默认管理员账号:
   - 用户名: `admin`
   - 密码: `admin123`
3. 登录成功后进入主控制台。

## 2. 主界面说明
- 左侧导航栏:
  - 仪表盘
  - 楼层监控
  - 三维监控
  - 分组控制
  - 情景模式
  - 定时调度
  - 能耗统计
  - 系统设置
- 顶部栏:
  - 当前页面标题
  - 当前登录用户
  - 退出登录

## 3. 仪表盘
- 查看楼层数、照明回路总数、当前开启数量和实时功率。
- 查看最近 7 日能耗趋势图。
- 查看最近操作日志。
- 查看近 30 日各楼层能耗对比图。

## 4. 楼层监控
- 左侧选择楼层。
- 中央区域查看楼层平面图。
- 不同房间按区域块展示，不同回路按灯位节点展示。
- 点击灯位后，可在右侧查看:
  - 状态
  - 地址
  - 电流
  - 功率
  - 亮度
- 可执行操作:
  - 单回路开关控制
  - 单回路亮度调整
  - 当前楼层全开
  - 当前楼层全关

## 5. 三维监控
- 支持显示全部楼层或聚焦单层。
- 三维视图中的球体表示灯位状态。
- 右侧列表可快速定位当前可见回路。
- 选中回路后可直接进行:
  - 开关切换
  - 亮度设置
- 点击“重置视角”可恢复默认观察角度。

## 6. 分组控制
- 仅管理员可创建、修改、删除分组。
- 操作步骤:
  1. 在右侧填写分组名称和描述。
  2. 勾选需要加入分组的回路。
  3. 点击“保存分组”。
- 对已有分组可执行:
  - 编辑
  - 开启
  - 关闭
  - 删除
  - 统一亮度设置

## 7. 情景模式
- 系统内置预设模式，如全开、全关、下班模式、节能模式。
- 仅管理员可创建和编辑自定义情景模式。
- 创建方式:
  1. 输入模式名称和描述。
  2. 为每个选中回路设置目标开关状态与目标亮度。
  3. 点击“保存模式”。
- 任意用户可应用情景模式。

## 8. 定时调度
- 仅管理员可创建、修改、删除定时任务。
- 支持四类目标:
  - 单回路
  - 分组
  - 情景模式
  - 全楼
- 创建步骤:
  1. 输入任务名称。
  2. 填写 Cron 表达式。
  3. 选择目标类型与目标对象。
  4. 设置目标状态和目标亮度。
  5. 点击“保存任务”。
- 页面支持查看:
  - 最近执行时间
  - 是否启用
  - 最近执行记录

## 9. 能耗统计
- 选择统计开始日期和结束日期。
- 查看:
  - 总能耗
  - 日能耗趋势图
  - 楼层能耗柱状图
  - 回路能耗排行
- 点击“导出CSV”可生成本地报表。

## 10. 系统设置
- 仅管理员可修改通信配置和密码。
- 支持两类通信模式:
  - 模拟器
  - Modbus TCP
- 可配置内容:
  - 主机地址
  - 端口
  - UnitId
  - 线圈/寄存器基地址
  - 采样周期

## 11. 日志查询
- 可按起止日期筛选操作日志。
- 支持分页浏览。
- 日志记录系统中的关键控制与调度行为。

## 12. 注意事项
- 非 Windows 环境仅用于构建与测试，桌面界面验收请在 Windows 上完成。
- 若使用 Modbus TCP 真机模式，需根据实际控制器寄存器映射调整通信配置。
- 默认演示建议优先使用模拟器模式。
